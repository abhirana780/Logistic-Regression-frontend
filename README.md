# AI Strategy Suite: Logistic Regression (Frontend) 💎

An interactive, high-fidelity AI laboratory designed for MBA students and business leaders to master Logistic Regression.

## 🌟 Strategic Features

- **4-Phase AI Lifecycle:**
  1. **Data Sourcing:** Connect to the raw business data.
  2. **Model Construction:** Select independent variables (Levers).
  3. **Strategic Analytics:** Validate Accuracy, Sensitivity, and ROI.
  4. **Risk Simulator:** Run "What-If" scenarios with a Strategic Verdict.
- **Horizon UI Design:** Premium, glassmorphic aesthetic with dark-mode analytics cards.
- **Dynamic Decision Zones:** Visual scatter plots showing data separation and boundaries.
- **Equation Decoder:** Peeks "under the hood" to show the exact math driving the AI.

## 🚀 Getting Started

1. Ensure the backend is running on `http://localhost:5000`.
2. Open `index.html` in any modern browser.
3. Pulse through the steps to see the AI in action!

## 🎨 Design System

- **Typography:** Outfit & Inter (Google Fonts)
- **Visuals:** Chart.js, Vanilla CSS Glassmorphism
- **UX:** F-Pattern alignment for executive reading.
